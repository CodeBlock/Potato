Welcome to Potato!

Potato is a new, minimalistic web framework for the Io programming language.

In its simplest form, a Potato application looks like this:

app := Potato clone

app GET("/", method(
  return app ok("Hello world!")
))

app run

PotatoSkin, Potato's tempating system, lets you separate logic from presentation. Here's a minimalistic template: 

Hello $name!

There are two ways to print evaluated Io code/variables with PotatoSkin. $variable works for simple variables that are in the scope of the template; and ${= SomeObject variable} works for accessing variables embedded within objects. Of course, you can evaluate Io in your template, too: 

<h1>Hello $name!</h1>
<i>Did you know that 2 + 2 is ${= 2+2}?</i>
${
  foo := "bar"
  dog := Object clone do (
    name := "Barky"
  )
}
My dog's name is <b>${= dog name}</b>.
